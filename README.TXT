mailSPAM это очень простой в использовании спамер.
С помощью данного приложения вы можете:
1.отомстить недругу.
2.подшутить над другом.
Чтобы установить файл вам потребуется:
1.Скачать Termux.
2.Вписать комманду "apt update&apt upgrade".
3.Вписать комманду "apt install git -y".
4.Установить Python коммандой "pkg install python -y".
5.Скопировать ссылку репозитория. 
6.Вписать комманду "git clone" и вставить скопированную ссылку (если просит логин и пороль, надо в конце ссылки вставить ".git").
7.Открыть только что скопированный репозиторий коммандой "cd MailSpam".
8.Открыть файл "mailSPAM.py" коммандой "python mailSPAM.py".
9.Указать данные которые просят.
